In the videos above, one is embeded from YouTube and the other uses a video file with a specific format(s).

The first video (binary numbers) is "streaming" from a local file, which is stored in the websites directory, accessed by a relative path, using video tags.

The second video, the YouTube video, is embedded in iframe tags, and uses an absolute URL.

Both videos require attributes such as width, height, and src, but they use different tags/elements to achieve this.

The html video element has nested source tags to specify the source (usually more than 1 format is available), whereas the YouTube video uses an iframe element to hold the src attribute.

Each of the tags also uses some of their own unique attributes, depending on if it is a YouTube video using an iframe, or uses the video tags (for a file like .mp4, .mov, etc).
